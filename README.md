# WeatherForecast
一款简单的天气预报App
>该项目是使用Kotlin语言编写，使用了ViewModel，LiveData和ViewBing，MutiType搭建的UI框架与及Retrofit作为网络的请求框架

>主要功能是显示今天的天气信息，与及未来六天的天气信息和最高温度，最低温度的折线图变化
